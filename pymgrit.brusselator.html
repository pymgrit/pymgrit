class pymgrit.brusselator.brusselator.Brusselator(*args, **kwargs)¶

Bases: pymgrit.core.application.Application

Application class for Brusselator system,

x’ = A + x^2y - (B + 1)x, y’ = Bx - x^2y,

with A = 1, B = 3, and ICs

x(0) = 0, y(0) = 1
